# Imagens de Golpes Especiais

Este diretório contém as imagens dos golpes especiais cadastrados no sistema.

## Especificações

- **Formatos aceitos**: JPG, PNG, GIF, WebP
- **Tamanho máximo**: 2MB
- **Dimensões recomendadas**: 400x400px ou similar (quadrado)
- **Nomenclatura**: golpe_{timestamp}_{uniqid}.{extensão}

## Upload

As imagens são enviadas através do painel administrativo em:
`Admin > Gerenciamento RPG > Golpes Especiais`

## Exemplos de uso

- Técnicas de ninjutsu (Katon, Suiton, etc.)
- Golpes de artes marciais
- Magias ofensivas/defensivas
- Habilidades especiais de classes
