# Diretório de Imagens de Habilidades

Este diretório armazena as imagens das habilidades disponíveis no sistema RPG.

## Especificações
- **Tamanho máximo**: 2MB por imagem
- **Formatos aceitos**: JPG, PNG, GIF, SVG
- **Nomenclatura**: `habilidade_{id_unico}.{extensão}`

## Upload
As imagens são enviadas através do painel administrativo em:
`/admin/content-management-rpg/habilidades`

## Limpeza
- Imagens são automaticamente deletadas quando a habilidade é removida
- Imagens antigas são substituídas quando uma nova é enviada para a mesma habilidade
