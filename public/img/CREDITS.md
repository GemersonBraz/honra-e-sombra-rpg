# Créditos dos Ícones

Este projeto utiliza ícones da biblioteca **game-icons.net**, uma coleção gratuita de ícones para jogos sob licença Creative Commons.

## Licença

Todos os ícones utilizados neste projeto estão sob a licença **CC BY 3.0** (Creative Commons Attribution 3.0).

- **Fonte**: https://game-icons.net
- **Licença**: https://creativecommons.org/licenses/by/3.0/
- **Requisito**: Atribuição aos criadores

## Principais Artistas

Os ícones foram criados pelos seguintes artistas:

- **Lorc** - Criador de muitos ícones utilizados
- **Delapouite** - Diversos ícones temáticos  
- **Carl Olsen** - Contribuições variadas
- **Sbed** - Ícones específicos
- **Faithtoken** - Elementos decorativos
- **Outros colaboradores** - Lista completa em https://game-icons.net/about.html#authors
